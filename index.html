HowFar

"How Far” is a sleek, real-time chat application built to showcase WebSocket-based communication with a modern and responsive UI. The backend is powered by FastAPI, enabling live messaging, user join/leave notifications, an online users counter, and message timestamps. On the front-end side, a lightweight HTML/CSS/JavaScript client connects to the WebSocket endpoint, allowing multiple users to chat instantly and seamlessly across devices. The design is mobile-friendly, fast and minimal, making it ideal both as a demonstration of real-time web communication and as a foundation for more advanced features such as authentication, private rooms or message persistence.
